# Services officiels et demarches

Ce document resume des informations generales sur les plateformes et services officiels cites par les sources publiques consultees.

## Plateforme officielle

Les sources officielles consultes renvoient vers la plateforme Nusuk pour consulter des informations utiles sur:

- les parcours de pelerinage;
- certaines demarches de reservation;
- des guides de sensibilisation;
- des informations pratiques pour les visiteurs et pelerins.

## Hajj

Pour le Hajj, les informations officielles publiques consultees insistent sur le fait qu'il faut suivre les circuits et plateformes autorises. Le chatbot doit eviter de promettre qu'un simple agent ou service non officiel suffit pour les demarches.

## Omra

Pour la Omra, l'utilisateur doit verifier:

- les conditions de voyage;
- les exigences sanitaires en vigueur;
- les informations officielles de la saison;
- les consignes publiees par les autorites competentes.

## Regle importante pour le chatbot

Le chatbot ne doit pas:

- inventer des regles de visa;
- garantir une admissibilite;
- donner des assurances juridiques;
- contredire une consigne officielle recente.

Le chatbot peut:

- orienter vers les plateformes officielles;
- rappeler de verifier les conditions a jour;
- expliquer les etapes generales de preparation.
